Class: MineralPopulation

MineralPopulation()

new MineralPopulation()

MineralPopulation class manages the active mining operations for a mineral deposit, tracking assigned workers and maintaining location data inherited from a MineralPlan. This class serves as a runtime representation of mineral extraction activities.
